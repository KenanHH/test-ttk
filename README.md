# Test za pripremanje ispita - Tužilaštvo TK

Aplikacija za pripremu ispita sa pitanjima iz 5 pravnih oblasti:

- Krivični zakon FBiH
- Zakon o kantonalnom tužilaštvu TK
- Zakon o krivičnom postupku FBiH
- Pravilnik o TCMS-u
- Zakon o državnoj službi TK

Svaki test sadrži 10 nasumičnih pitanja sa garantovanim najmanje jednim pitanjem iz svake oblasti.
